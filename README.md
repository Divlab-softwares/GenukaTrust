# GenukaTrust
Genuka Trust est une application intelligente connectée à la plateforme Genuka, conçue pour automatiser entièrement la collecte, la gestion et la valorisation des avis clients.Après la livraison d’une commande, l’application envoie automatiquement un SMS ou un message WhatsApp au client afin de recueillir une note (1 à 5 étoiles) et un commentaire.
